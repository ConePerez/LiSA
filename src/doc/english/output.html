Output of schedules

As soon as LiSA has computed a schedule it will display it in the main window.

How to change view:

Choose your favourite output format in the menue View. There are implemented:
  • Sequence - latin rectangle according to the block matrices model
  • Sequence Graph - graph according to the disjunctive graph model
  • Schedule - matrix of operation completion times
  • Gantt-Chart
  • furthermore you can display the problem instance in the main window
