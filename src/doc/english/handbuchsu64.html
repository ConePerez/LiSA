<integer>

This declares an integer parameter that can be passed to the algorithm.

Parent element: <alg_controls>

Child elements: none

Attribute

Meaning

 

name

Name of the parameter. With this name, the parameter is identified internally.

 

caption

The name of the parameter in a human readable form. This name is shown in the parameter window when the algorithm is called via the menu.

 

default

The standard value for this parameter.
