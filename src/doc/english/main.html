What is LiSA?

LiSA - Library of Scheduling Algorithms - is a software for solving and examining scheduling problems. Within a user-friendly graphical environment there are algorithms as well as tools for displaying and manipulating schedules. LiSA is intended to be used in research for fast implementation and testing of algorithms, in teaching for examining properties of existing algorithms and it supports the developement of scheduling algorithms for practical use.

Elementary Usage

To obtain a solution for a scheduling problem, please follow these steps:
  • enter a problem type
  • look up the complexity of the problem
  • enter the parameters defining the problem instance
  • call a solution algorithm
  • display schedule
  • manipulation of schedule

Further Features

  • testing parameters of algortihms
  • numerical evaluation
  • integration of external algorithms

Where do I get more Information?

Information about the newest LiSA-Version and the block matrices model which is the basis of diplaying schedules in LiSA is available on the LiSA-Homepage . For questions regarding implementation of algorithms for LiSA please consult the developer's manual.
