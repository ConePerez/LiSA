Chapter 3
Input

An instance of a problem that shall be solved with LiSA consists of the problem type (→3.1), the number of jobs n, the number of machines m and the parameters (→3.2) that are necessarily related to the problem type, n and m. The input can be done by hand or by opening an input file. In addition, LiSA offers a random generator that generates uniformly distributed data from a given interval. To start the generator, two random integer values, the time seed and the machine seed, have to be given. They can be either given by hand or they are automatically set to the current system time. Entering them manually allows the recreation of an already generated problem.
