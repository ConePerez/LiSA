<release_times>

This element contains the release dates of the jobs as an n-dimensional vector. The i-th component of this vector contains the release date of job Ai.

Parent element: <values>

Child elements: A vector with n components.

Attribute

Meaning

 

model

Reserved, must be lisa_native.
