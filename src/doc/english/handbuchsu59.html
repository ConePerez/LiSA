<problem>

This element represents a problem type that can be solved. It is either placed in the <exact> or <heuristic> elements and contains each <alpha>, <beta> and <gamma> elements that specify the problem type in the common α∣β∣γ notation.

Parent element: <heuristic> or <exact>

Child elements: <alpha>, <beta>, <gamma>

This element does not contain attributes.
