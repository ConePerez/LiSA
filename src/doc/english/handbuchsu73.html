A.4.3 LiSA-Program Parameters

Type

Name

Meaning

integer

WIDTH

Width of the LiSA window at the program startup (in pixels).

integer

HEIGHT

Height of the LiSA window at the program startup (in pixels).

string

STARTFILE

File name of a problem in XML format that is initially shown at the program startup.

string

HTML_VIEWER

The console command that LiSA should use to start the standard browser, for example iexplore (Windows), konqueror oder firefox (Linux)

string

LANGUAGE

Here the program language can be selected (either english or german).
