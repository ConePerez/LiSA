<gamma>

γ describes the objective function.

Parent element: <problem>

Child elements: none

Attribute

Meaning

 

objective

The objective function of the problem. Possible values are Cmax, Lmax, Sum_Ci, Sum_wiCi, Sum_Ui, Sum_wiUi, Sum_Ti, Sum_wiTi, irreg_1, irreg_2
