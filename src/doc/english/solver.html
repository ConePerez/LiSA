Algorithms in LiSA

To obtain an exact or heuristic solution for your problem instance you can call one of the implemented algorithms.

How to call an algorithm:

Make sure both a problem type all problem parameters are defined. Then the menu Algorithms lists all implemented algorithms which can be applied to this problem type. Choose one, edit parameters if necessary and click OK.

Implemented algorithmens

  • branch and bound
  • priority dispatching rules
  • neighborhood search

Exceptions

  • For some problem types there is no implemented algorithm. In this case, tha algorithm menue is empty.
  • User-own external algorithms have to be integrated into LiSA to be available within the menue.
