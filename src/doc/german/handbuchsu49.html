<parameter>

Dieses Element stellt einen Parameter zur Übergabe an einen Algorithmus dar.

Elternelement: <controls>

Kindelemente: keine

Attribut

Bedeutung

 

type

Der Datentyp des Parameters. Mögliche Werte sind string, integer, real

 

name

Der Name des Parameters. Eine Liste der Algorithmen und deren zugehöriger Parameter findet sich im Kapitel 4.

 

value

Der eigentliche Wert des Parameters.
