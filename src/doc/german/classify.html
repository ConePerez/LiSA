Problemklassifikation für Schedulingprobleme

Benutzen Sie diesen Menüpunkt, um das aktuell unter dem Menüpunkt Problemtyp eingestellte Problem (basierend auf der sogenannten 3-Feld Notation) bezüglich der Komplexität seiner Berechenbarkeit zu klassifizieren.

Diese Klassifikation wird durch Analyse einer BibTeX-Datenbank (Datei classify.bib) durchgeführt, die auf folgender Sammlung von Resultaten für Schedulingprobleme basiert: Complexity results of scheduling problems.

So kommen Sie hierher:

Stellen Sie sicher, daß ein Problemtyp definiert ist. Wählen Sie dann den Menüpunkt Extras|Problemklassifikation.

Ausgabe im Problemklassifikations - Fenster:

  • Angabe, ob das Problem

    • polynomial lösbar,
    • pseudo-polynomial lösbar,
    • NP-schwer oder
    • NP-schwer im strengen Sinne ist.

  • Wenn die Komplexität des betrachteten Problems bekannt ist, wird ein Hinweis auf die entsprechende Literaturquelle gegeben, die die Aussage belegt.

  • Für die Ausgabe der entsprechenden vollständigen Literaturquellen, drücken Sie den Button Vollständige Literaturquellen.
