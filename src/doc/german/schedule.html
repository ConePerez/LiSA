Das Schedule-Fenster

Benutzen Sie dieses Fenster zur Darstellung und Manipulation einer Lösung in Matrixform. Es kann ebenfalls gewählt werden, ob stets eine semiaktive Lösung vorliegen soll. Dieses Fenster ist anwählbar, falls alle Parameter, die zur Problembeschreibung gebraucht werden, vollständig eingegeben wurden.

So kommen Sie hierher:

Stellen Sie sicher, daß ein Problemtyp und alle Problemparameter definiert sind. Wählen Sie dann den Menüpunkt Bearbeiten|Lösung

Allgemeines:

Ein Schedule, das die Eigenschaft besitzt, daß keine Operation eher ausgeführt werden kann, ohne den Start mindestens einer weiteren Operation zu verzögern, wird semiaktives Schedule genannt. Das Modell des Lateinischen Rechtecks eignet sich sehr gut zur Darstellung einer semiaktiven Lösung in Matrixform. Weiterführende Informationen zu diesem Modell findet man unter: Latin Rectangles in Scheduling Theory - a Basic Modelling Concept. Wichtig in diesem Zusammenhang ist, daß zu jeder Wahl von Problemparametern stets ein optimales semiaktives Schedule existiert, falls eine reguläre Zielfunktion ausgewählt wurde.

Wenn Manipulationen nicht möglich sind:

  • Falls semiaktiv gewählt wurde, kann das Schedule nicht editiert werden. Benutzen sie in diesem Fall den Menüpunkt: Erzeugen|Als semiaktiven Plan festlegen um den Plan als nicht semiaktiv zu markieren.
