<problem>

Dieses Element stellt einen Problemtyp dar. Es enthält je einen <alpha>, <beta> und <gamma>-Tag, die den Problemtyp in der gängigen α∣β∣γ – Notation darstellen.

Elternelement:

  • keines im Dokumenttyp problem (Wurzelelement)
  • <instance> im Dokumenttyp instance
  • <solution> im Dokumenttyp solution

Kindelemente: <alpha>, <beta>, <gamma>

Dieses Element besitzt keine Attribute.
