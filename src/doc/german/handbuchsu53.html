<algorithm>

Dieses Element umfasst die gesamte Algorithmenbeschreibung.

Elternelement: keines (Wurzelelement des algorithm-Dokumentes).

Kindelemente: <heuristic>, <exact>, <alg_controls> (optional).

Attribut

Bedeutung

 

name

Name des Algorithmus, wie er in LiSA im Menü Algorithmen angezeigt wird.

 

type

Entweder constructive oder iterative. Iterative Algorithmen benötigen eine bereits vorher durch einen konstruktiven Algorithmus erstellte Lösung, um darauf aufzubauen.

 

code

Reserviert, hier gibt es nur die Möglichkeit external

 

call

Name des Programms, das den Algorithmus implementiert, ohne Dateierweiterung (z.B. .exe). Dieser stimmt normalerweise mit dem Namen des Ordners überein, in dem der Quellcode des Algorithmus liegt.

 

help_file

Name der HTML-Hilfedatei (mit Dateierweiterung, z.B. .html).
