<integer>

Deklariert einen ganzzahligen Parameter, der dem Algorithmus übergeben werden kann.

Elternelement: <alg_controls>

Kindelemente: keine

Attribut

Bedeutung

 

name

Der Name des Parameters, mit dem er intern angesprochen wird.

 

caption

Der Name des Parameters, in einer menschenlesbaren Form. Dies ist die Bezeichnung, wie sie im Parametereingabefenster nach der Auswahl Algorithmus dargestellt wird.

 

default

Der Standardwert für diesen Parameter.
