Darstellung von Plänen

Sobald LiSA einen Plan berechnet hat oder Sie einen Plan per Tastatur oder Datei eingegeben haben, wird dieser im Hauptfenster dargestellt.

So wechseln Sie die Darstellung:

Wählen Sie die entsprechende Darstellung im Menü Ansicht. Implementiert sind:
  • Plan - die Darstellung als lateinisches Rechteck
  • Plangraph - die Darstellung im disjunktiven Graphenmodell. Bei C_max-Problemen erscheint der kritische Weg auf Wunsch anderfarbig.
  • Schedule - die Matrix der Bearbeitungsendzeitpunkte
  • Gantt-Diagramm
  • außerdem lassen sich die Werte im Hauptfenster anzeigen.
