A.4.3 LiSA-Programmparameter

Typ

Name

Bedeutung

integer

WIDTH

Fensterbreite von LiSA beim Programmstart (in Pixel).

integer

HEIGHT

Fensterhöhe von LiSA beim Programmstart (in Pixel).

string

STARTFILE

Dateiname eines Problems in XML-Format, das beim Programmstart von LiSA automatisch geladen werden soll.

string

HTML_VIEWER

Der Konsolenbefehl, den LiSA benutzen soll, um den standard-HTML-Browser zu starten, z.B. iexplore (Windows), konqueror oder firefox (Linux)

string

LANGUAGE

Hier kann die Programmsprache ausgewählt werden (entweder english oder german).
