<parameter>

Dieses Element stellt einen Programmparameter zur Übergabe an LiSA dar.

Elternelement: <controls>

Kindelemente: keine

Attribut

Bedeutung

 

type

Der Datentyp des Parameters. Mögliche Werte sind string, integer, real

 

name

Der Name des Parameters.

 

value

Der eigentliche Wert des Parameters.
