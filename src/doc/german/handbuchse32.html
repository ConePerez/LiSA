A.2 Die Dokumenttypen problem, instance und solution

Diese drei Dokumenttypen dienen allgemein zur Kommunikation von Schedulingproblemen in LiSA. Sie bauen Schrittweise aufeinander auf. So speichert ein problem-Dokument nur einen Problemtyp in α∣β∣γ – Notation. Ein instance-Dokument speichert zusätzlich eine konkrete Probleminstanz und ein solution-Dokument eine Probleminstanz zusammen mit einer oder mehrerer Lösungen.
