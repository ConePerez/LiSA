Eingabe der Werte

Hier bestimmen Sie die Parameter Ihrer Probleminstanz. Welche Parameter angezeigt bzw. editiert werden, kann durch Auswahl des entsprechenden Menüeintrages unter Ansicht bestimmt werden. Klickt man auf ein Feld in der angezeigten Tabelle, wird dieses editiert. Weiterhin ist es möglich, alle Parameterwerte zufällig zu erzeugen.

So kommen Sie hierher:

Stellen Sie sicher, daß ein Problemtyp definiert ist und wählen Sie den Menüpunkt Bearbeiten|Werte.

Eingabemöglichkeiten

Abhängig vom gewählten Problemtype können einige der folgenden Parameter bearbeitet oder generiert werden:
Bearbeitungszeiten:
die Zeit, die eine Operation zu ihrer Ausführung benötigt
Operationenmenge:
Eintrag in Zeile i und Spalte j dieser Matrix ist 1, wenn die Operation (i,j) existiert, 0 sonst
Technologische Reihenfolge:
Eine Permutation der Zahlen 1 bis m in jeder Zeile i bestimmt die Reihenfolge, in der in der Auftrag i auf den einzelnen Maschinen bearbeitet wird. Soll z.B. Auftrag 3 zuerst auf Maschine 2 und dann auf Maschine 5 bearbeitet werden, so steht in Zeile 3, Spalte 2 eine 1 und in Spalte 5 eine 2. Nach dem Editieren muß der Button Technologien übernehmen betätigt werden.
Fälligkeitstermine:
ein Zeitpunkt, zu dem der jeweilige Auftrag fertiggestellt sein soll
Gewichte:
Wichtungen der Aufträge in bestimmten Zielfunktionen
Bereitstellungszeiten:
ein Zeitpunkt r_i, zu dem Auftrag i fr$uuml;hestens bearbeitet werden kann.

Ausnahmen

  • Sie können nur solche Parameter editieren, die in Problemen des gewählten Problemtyps vorkommen.
  • Technologien werden nur dann übernommen, wenn die gesamte Matrix korrekt ausgefüllt ist.
