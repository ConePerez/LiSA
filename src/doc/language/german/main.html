Wozu LiSA?

LiSA - Library of Scheduling Algorithms - ist eine Software zur Lösung und Untersuchung von Scheduling-Problemen. Unter einer benutzerfreundlichen Oberfläche finden sich verschiedene Algorithmen sowie Veranschaulichungen und Manipulationstools für Pläne. Die Einbindung nutzereigener Algorithmen in die Anwendung ist extrem komfortabel möglich.

Elementare Benutzung

Um eine geignete Lösung eines Scheduling-Problems zu erhalten, führen Sie im wesentlichen folgende Schritte aus:
  • Eingabe des Problemtyps
  • Problemklassifikation
  • Eingabe der Probleminstanz
  • Aufruf eines Algorithmus
  • Darstellung von Lösungen
  • Planmanipulation

Weiterere Möglichkeiten

  • Anpassen von Parametern
  • Auswerten von numerischen Ergebnissen
  • Einbinden nutzereigener Algorithmen
