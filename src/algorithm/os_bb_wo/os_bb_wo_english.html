Brucker's Open-Shop B&B

This algorithm was designed and implemented at the university of Osnabrück by the group of Peter Brucker. It can solve problems of the type O||Cmax exactly.

What you have to do::

Make sure that both a problem type and all parameters are defined. Choose Exact Algorithms|Open-Shop B&B from the menu Algorithms and after editing the options press OK. You may stop the algorithm at any time, it will then return the best schedule found so far.

Options:

Heuristic
The algorithm can use various heuristics to construct/complete schedules in each branch node. There are more heuristics than those listed here, they have however been disabled since they don't seem to work with incomplete operation sets.
LB_PREC_RULE
A heuristic based on a precedence rule determined by certain lower bounds.
LB_PREC_RULE_VAR
A variation of the rule above.
