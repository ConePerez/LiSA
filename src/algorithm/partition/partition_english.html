partition Algorithm

Use this algorithm to solve problems of the type P2 // Cmax.
It is based on a pseudopolynomial algorithm for the PARTITION problem, wich uses the principle of dynamic programming. This implementation was done in 2001 bei Claudia Isensee.

This is, how You get here:

Make sure, that You have specified Problem Type (P2 // Cmax) and Parameters. Choose the menu item Algorithms|Exact Algorithms|partition-module.

Parameters

The problem requires only one pi per job. Right now there is still offered a two column matrix. It is sufficient to fill in the first column of the matrix.

Problems:

  • A problem of the type P2 // Cmax does not yet offer partition as an algorithm, therefore You have to camouflage you problem as O2 // Cmax.
  • Displaying the solution is only possible through opening ~/.lisa/proc/algo_out.lsa, for now.
