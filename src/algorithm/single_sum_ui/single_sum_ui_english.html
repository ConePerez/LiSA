sample Algorithmus

Use this algorithm to solve an one-machine problem where the total number of delayed jobs has to be minimized. This algorithm always finds an optimal solution.

What you have to do::

Make sure that both a problem type and all parameters (processing times and due dates) are defined.

Options:

No parameters used.

Problems:

  • auto_alg currently does not accept one-machine problems. Use the problem type O / / SumUi instead.
