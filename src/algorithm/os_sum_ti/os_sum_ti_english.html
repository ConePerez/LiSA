os_sum_ti algorithm

U can use this algorithm to solve an open shop problem with SumTi objective heuristically. This algorithm is designed to solve problems with unit processing times exact, with no-wait criteria also.

What you have to do:

Make sure that both a problem type and all parameters are defined. Choose Exact Algorithms|os_sum_ti or Heuristic Algorithms|os_sum_ti from the menu Algorithms and after editing the options press OK.

Problems:

  • The exact algorithm is laid out for complete operation sets.
