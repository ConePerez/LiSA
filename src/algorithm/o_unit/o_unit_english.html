o_unit algorithm

Use this algorithm to solve a open shop problem with Cmax,SumCi objective heuristically. This algorithm is for problems with unit processing times exact.

What you have to do:

Make sure that both a problem type and all parameters are defined. Choose Exact Algorithms|o_unit from the menu Algorithms and after editing the options press OK.

Options:

seed parameter
the seed parameter is used to initiate the random function in the algorithm.

Problems:

  • The result of this algorithm is for some O/p_ij=1/Cmax,SumCi problems with incomplete operation set not optimal.
