Lösungsalgorithmen in LiSA

Um eine exakte oder näherungsweise Lösung für die aktuelle Probleminstanz zu erhalten, können Sie einen der implementierten Algorithmen benutzen.

So rufen Sie einen Algorithmus auf:

Stellen Sie sicher, daß ein Problemtyp und alle Problemparameter definiert sind. Im Menü Algorithmen werden dann automatisch diejenigen implementierten Algorithmen aufgelistet, die für den aktuellen Problemtyp anwendbar sind. Wählen Sie einen der Punkte aus, editieren Sie gegebenenfalls einige Parameter und drücken Sie dann OK.

Implementierte Algorithmen

  • Branch and Bound
  • Reihungsregeln
  • Nachbarschaftssuche

Ausnahmen

  • Für manche Problemtypen ist eventuell keiner der implementierten Algorithmen anwendbar. In diesem Fall bleibt das Menü leer und kann nicht geöffnet werden.
  • Nutzereigene externe Algorithmen müssen vor dem Aufruf in LiSA eingebunden werden.
