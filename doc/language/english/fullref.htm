Output of the complete references during the problem classification

Please use this window for displaying the complete references as BibTeX entries according to the reference list in the Problem Classification window.

How to come here:

Make sure that you have defined a Problem Type. Choose the menu entry Extras|Problem Classification|Complete References.

Remarks on the BibTeX entries:

  • Each key of a database entry consists of the corresponding Mathematical Reviews Number (e.g. MR22:1461) or Zentralblatt Number (e.g. 184.23303), if available.

  • Otherwise the key is built from the author(s) and the publication year (e.g. baker:1974).

  • The abbreviations for the classes in the annotation records of the BibTeX file classify.bib are as follows.

    Complexity ClassMeaning
    $P$polynomially solvable
    $P_{pseudo}$pseudo-polynomially solvable
    $\NP$-hard$\NP$-hard (in the ordinary sense)
    $*\NP$-hard$$\NP$-hard in the strong sense
